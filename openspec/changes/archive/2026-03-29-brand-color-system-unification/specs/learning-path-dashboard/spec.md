# learning-path-dashboard (Delta Spec)

## MODIFIED

- **REQ-DASHBOARD-BRAND-CONSISTENCY**: Dashboard badges, stat cards, topic headers, topic cards, chips, and highlight surfaces SHALL use the unified FluenZ brand token system instead of off-brand orange/beige accents.
- **REQ-DASHBOARD-SUBTLE-HIGHLIGHTS**: Dashboard highlight surfaces SHALL use soft cyan, blue, or violet tints to preserve hierarchy without introducing noisy or off-brand emphasis.

### Scenario: Dashboard follows one visual language
- **WHEN** a user opens `/dashboard`
- **THEN** cards, badges, chips, borders, and selected states feel visually unified
- **AND** the page reflects the FluenZ teal-cyan-blue-violet brand family
