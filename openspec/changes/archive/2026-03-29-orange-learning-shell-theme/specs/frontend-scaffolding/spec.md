# frontend-scaffolding (Delta Spec)

## MODIFIED

- **REQ-THEME-TOKENS**: The frontend SHALL expose semantic theme tokens for branded surfaces, borders, text, and accents instead of depending on page-level hardcoded colors for the learning experience.
- **REQ-THEME-MODES**: The frontend SHALL provide token values for both light mode and dark mode.
- **REQ-BRAND-CONFIG-READY**: The primary accent used by learning screens SHALL be centrally defined so it can be replaced later by admin-managed configuration without rewriting each screen.
