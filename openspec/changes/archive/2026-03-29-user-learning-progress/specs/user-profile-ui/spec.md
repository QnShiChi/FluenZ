## ADDED Requirements

### Requirement: User Profile Dashboard
The system MUST provide a central Profile screen where users can view their learning statistics.

#### Scenario: Navigating to Profile
- **WHEN** the user clicks the "Hồ sơ" icon in the left Navigation sidebar
- **THEN** they are directed to a Profile page showing their Avatar, today's learning duration, 5-minute daily goal progress, weekly learning calendar, current streak, and the "100 Hours" motivational card

### Requirement: Motivational Statistics
The system MUST display aggregated motivational statistics reflecting the user's total effort.

#### Scenario: Viewing total spoken count
- **WHEN** the user is viewing their Profile
- **THEN** the system MUST display the total number of times they have spoken English (total voice recordings submitted) across all practice sessions
